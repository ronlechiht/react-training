### How to start

- Clone project : 
```
git clone https://github.com/ronlechiht/react-training.git
```
- Move in practice folder: 
```
cd react-training/practice/e-commerce-website
```
- Checkout branch : 
```
git checkout dev
```
- Install packages: 
```
npm install
```
- Start project: 
```
npm run dev
```
- Open on web: http://localhost:5173
